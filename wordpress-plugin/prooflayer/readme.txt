Display beautiful testimonials and social proof on your WordPress site with ProofLayer.

== Description ==

ProofLayer makes it easy to collect and display customer testimonials on your WordPress website. Boost conversions with authentic social proof.

**Features:**

* Multiple layouts: Grid, List, Carousel, Masonry, Marquee, Spotlight
* Light and dark themes
* Animation effects: Fade, Slide, Floating Hearts
* Gutenberg block support
* Shortcode support
* Widget support for sidebars
* Mobile responsive
* Fast and lightweight

**How It Works:**

1. Sign up for a free ProofLayer account at [prooflayer.app](https://www.prooflayer.app)
2. Create a testimonial collection form
3. Share the form with your customers
4. Approve testimonials in your dashboard
5. Display them on your WordPress site with this plugin

== Installation ==

1. Upload the `prooflayer` folder to `/wp-content/plugins/`
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to Settings â†’ ProofLayer
4. Enter your Workspace ID (found in your ProofLayer dashboard)
5. Add testimonials using the Gutenberg block, shortcode, or widget

== Frequently Asked Questions ==

= Where do I find my Workspace ID? =

Log in to your ProofLayer dashboard at prooflayer.app, go to Widgets, and copy your Workspace ID from the embed code section.

= Can I use multiple widgets on one page? =

Yes! You can add multiple ProofLayer blocks or shortcodes with different settings on the same page.

= Does this work with page builders? =

Yes, the shortcode works with all major page builders including Elementor, Divi, and Beaver Builder.
